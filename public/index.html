Objetivo general

Al finalizar el módulo, tendrás las habilidades y conocimientos necesarios para crear una página web estática siguiendo buenas prácticas de desarrollo; además, comprenderás los fundamentos y herramientas de la Economía Monetaria y Financiera actuales, desde la teoría monetaria hasta la regulación de mercados y la gestión de riesgos.

Competencias a desarrollar

HTML & CSS

Domina semántica, BEM, Flexbox, Grid y Responsive Web Design.

Versionado con Git

Control de cambios, ramas y colaboración en GitHub.

Economía Financiera

Mercados, regulación y análisis monetario con enfoque contemporáneo.
